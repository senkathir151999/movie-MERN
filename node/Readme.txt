# Movie Database API

This Node.js project provides a RESTful API for managing a movie database, including movies, directors, and actors. The API allows you to perform CRUD (Create, Read, Update, Delete) operations on these entities.



## Getting Started

### Prerequisites

Before you can run the project, make sure you have the following prerequisites installed:

- Node.js: [https://nodejs.org/](https://nodejs.org/)
- MySQL: [https://www.mysql.com/](https://www.mysql.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd backend
   npm i
   npm start
